The list Item Element | <li>

The list item element was made specifically to be nested within an ordered list, unordered list, or less commonly, a menu. The list items in a menu and unordered lists render identically, they just have different semantic implications.

Unordered List Item Example:

Input:


<ul>
    <li>Ham</li>
    <li>Grilled Cheese</li>
    <li>Hot Dog</li>
</ul>

Output:

A list of Sandwiches
  • Ham
  • Grilled Cheese
  • Hot Dog

Ordered List Item Example:

Input:


<ol>
    <li>Reuben</li>
    <li>Chicken Salad</li>
    <li>Philly Cheesesteak</li>
</ol>

Output:

The Top Three Sandwiches
  1. Reuben
  2. Chicken Salad
  3. Philly Cheesesteak

Menu List Item Example:

Input:


<menu>
    <li>
        <button>Mayonnaise</button>
        <button>Ketchup</button>
        <button>Mustard</button>
    </li>
    <li>
        <button>Ham</button>
        <button>Turkey</button>
        <button>Bacon</button>
    </li>
    <li>
        <button>Cheese</button>
        <button>Lettuce</button>
        <button>Tomato</button>
    </li>

</menu>

Output:

Select Your Sandwich Toppings:
