The Header Element | <h1>

The header elements create a hierarchy of headlines that can rank from <h1>(most important) to <h6>(least important). Each header automatically applies font-weight: bold; and adjusts the size of the font.

It's important to note that header elements are content not containers. They do not create a new section and are subject to the box model of their parent element.

Sample Input and Output

Input:

<h1>H1</h1>

<h2>H2</h2>

<h3>H3</h3>

Output:

H1

H2

H3
