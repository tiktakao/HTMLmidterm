The Division Element | <div>

The div element is the most generic container. It's use has no particular semantic significance and is thus only suggested to be a container of last resort.

Sample Input

HTML:


    <div class="outterContainer">
        <div class="innerContainer">
        </div>
    </div>

CSS


    .outterContainer{
        border-style: solid;
        border-color: red;
        width: 100px;
        height: 100px;
    }

    .innerContainer{
        border-style: solid;
        border-color: green;
        width: 50px;
        height: 50px;
    }

Explanation:

In the HTML to the left, you see a div within a div. While this might not be the most semantically accurate example, it highlights this containers flexibility.

  • The parent div is a container that the CSS defines as a box 100 by 100 pixels, and outlined in red.
  • The child div is a container that the CSS defines as a box 50 by 50 pixels, and outlined in green.
