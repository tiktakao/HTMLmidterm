The Italic Element | <i>

The italic element is a text-level semantic element. This element italicizes the text, which creates a slanted appearance.

The italic element should not be confused with emphasis, which creates a similar effect, but is semantically more important.

Sample Input and Output

Input:

<i>This</i> text is really <i>SLANTED</i>.

Output:

This text is really SLANTED.
