The Meta Element | <meta>

The meta element is not direct content viewable on a page, but instead it provides metadata to the browser about the page. It's typically nested within the <head> of a page and is self-closing, meaning it does not require a <\meta>.

Some common uses for the meta tag include defining the character set, adding descriptors and keywords for search engine optimization, and defining settings for responsive web design.

Sample Input and Outcome

Input:

  1. <meta charset="UTF-8">
  2. <meta name="viewport" content="width=device-width, initial-scale=1.0">
  3. <meta name=description" content="View the latest news and breaking news today for U.S., world, weather, entertainment, politics and health at CNN.com.">
  4. <meta name="theme-color" content="#ffffff">

Outcome:

  1. This meta element defines the character set used on this page. Developers commonly use character set UTF-8, which covers most of the available characters available.
  2. This meta element is used for responsive web design and helps with scaling the page to be viewed on mobile devices.
  3. This meta element offers a description of the page. This particular description is from the CNN website.
  4. This meta element defines a theme color, which is a suggested color palette to be used by other developers who work on this site. This particular tag is from the CCRI website.
