The Cursor Properties | {cursor: }

The cursor is the little pointer you move around the screen to click on things™. As you might have noticed, this cursor can change its appearance contextually to give cues on what action it might perform if the user clicks the mouse. Using CSS we can dictate when and to what the pointer will change its appearance.

CSS Input:

  1. .button1{
    cursor: move;
}
                    
  2. .button2{
    cursor: crosshair;
}
                    
  3. 
.button3{
    cursor: wait;
}
                    
  4. 
.button4{
    cursor: help;
}
