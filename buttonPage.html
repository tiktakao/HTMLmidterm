The Button Element | <button>

The button element is an interactive element that creates a button which a developer can assign purpose. This purpose could be to submit information, reset a field, or trigger javascript. A button has a list of appendable attributes to affect its behaviour. For example:

  • Type: which allows the behavior of the button to be defined(submit, reset, etc.).
  • Autofocus: This makes the button the pre-selected field affected by the 'ENTER' key.
  • Disabled: Disables the button and makes it unclickable.
  • Onclick: triggers a JavaScript function.

Input:



    
    
  • <button autofocus>Autofocus</b>
    • 
    
  • <button type="submit">Submit</b>
    • 
    
  • <button disabled>Disabled</b>
    • 
    
  • <button onclick="alert('Yeahhhh thats a button')">...button?</b>
    •
