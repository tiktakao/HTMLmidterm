The List Style Type Property | {list-style-type}

The list-style-type property allows you to target the style of the list item. There are different styles accessible to the developer depending on whether the list item is nested within an ordered or unordered list. It is also possible to enter a custom character to be used.

While it is not directly related to the list-style-type, if you would like to further customize the style of the list marker, you can target the marker itself with pseudo-element "::marker." This can be seen in the third column where the marker has been colored red.

Unordered List Item Examples:

Input:


li{
    list-style-type: disc;
}

li{
    list-style-type: circle;
}

li{
    list-style-type: square;
}

Output:

  • Disc
  • Circle
  • Square

Ordered List Item Examples:

Input:


li{
    list-style-type: decimal;
}

li{
    list-style-type: upper-alpha;
}

li{
    list-style-type: upper-roman;
}

Output:

  1. Decimal
  2. Upper-Alpha
  3. Upper-Roman

Other List Item Examples:

Input:


li{
    list-style-type: "-";
}

li{
    list-style-type: "â™£";
}

li::marker{
    color: red;
}

Output:

  • Dash
  • Club
  • Red
