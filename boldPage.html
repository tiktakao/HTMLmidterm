The Bold Element | <b>

The Bold element is a text-level semantic element. It causes the targeted text to have an increased appearance of size and line width. How this element accomplishes this is by effecting the font-weight with a default value of 400, and increasing it to 700. It is used to highlight a word, without necessarily marking it as important.

An incorrect use of the Bold element would be to use it in place of a header. They may achieve a similar effect, but semantically denote two different things.

Sample Input and Output

Input:

<b>This</b> text is really <b>BOLD</b>.

Output:

This text is really BOLD.

But Don't Do This:

<b>But Don't Do This</b>:
