The Underline Element | <u>

The underline element is a text-level element with no semantic implication, and it's purpose is limited to labeling a text as being misspelled, a proper noun in Chinese text, or for using in stylistic reasons. However, the underline elements documentation offers the advice that this element should be avoided when there are more appropriate elements to choose from, and for some users, could be confused for a hyperlink.

Input: 


<u>This text is underlined, but not a hyperlink.</u>

Output:


This text is underlined, but not a hyperlink.

To learn more, visit the official W3 documentation here.
